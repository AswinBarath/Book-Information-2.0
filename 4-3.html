    • Description

    Grasp the fundamentals of web application development by building a simple database-backed app
    from scratch, using HTML, JavaScript, and other open source tools. Through hands-on tutorials,
    this practical guide shows inexperienced web app developers how to create a user interface,
    write a server, build client-server communication, and use a cloud-based service to deploy the application.
